os1 Implementation Rust Docs

os1 is a simple 32-bit x86 operating system written from scratch in Rust.

I had originally intended not to have any library code in my kernel at all, but this is not possible with Rust. As a result I am linking in `libcore` and `liballoc` which contain the heart of Rust functionality. Special thanks to Krzysztof for helping me get that set up!

The source is available on Github

Features so far:

  • Heap manager
  • Preemptive multitasking
  • VGA buffer management
  • Reaper and Idle processes
  • Concurrency Control (via Semaphores)
  • Virtual Memory

Requirements:

  • qemu
  • gcc
  • rust 1.6 (2015-11-08 nightly)

    You need to have nightly Rust (not Beta or release) because the project uses some unstable language features, such as libcore.

    curl -sSf https://static.rust-lang.org/rustup.sh | sh -s -- --channel=nightly --date=2015-11-08

Build instructions

  1. git clone https://github.com/mark-i-m/os1.git
  2. cd os1/
  3. git submodule init && git submodule update
  4. make rungraphic
